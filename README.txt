1. Extract TSModpack.rar

2. copy-paste contents of BepInExPack_Valheim folder into Valheim local files (replace if prompted)
(default: "SteamLibrary\steamapps\common\Valheim" in Steam right-click Valheim and select Manage>Browse Local Files to get there quickly)

3. run the game and make sure BepInEx version is displayed in the top left

4. exit the game

5. copy-paste contents of 'plugins' folder located in extracted "TSModpack\plugins" into "\steamapps\common\Valheim\BepInEx\Plugins"
(replace if prompted)

6. copy-paste contents of 'config' folder located in extracted "TSModpack\config" into "\steamapps\common\Valheim\BepInEx\config"
(replace if prompted)

7. enjoy
